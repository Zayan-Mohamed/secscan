docs: Add comprehensive MkDocs documentation site

This commit introduces a complete documentation system using MkDocs with
Material theme for the SecScan project.

Documentation Structure:
- Getting Started guides (Quick Start, Installation, First Scan)
- User Guide (Basic Usage, Configuration, CI/CD Integration, Examples)
- Reference documentation (CLI Options)
- Deployment guides and setup scripts

Key Features:
- Professional Material theme with dark/light mode support
- Fast, searchable documentation with built-in search
- Mobile-responsive design
- Code syntax highlighting with copy buttons
- Tabbed content for platform-specific instructions
- Admonitions for tips, warnings, and info boxes
- Automated deployment via GitHub Actions

New Files Added:
- mkdocs.yml - MkDocs configuration
- requirements-docs.txt - Python dependencies for documentation
- setup-docs.sh - Automated setup script
- DEPLOYMENT.md - Deployment guide for docs
- DOCS_GUIDE.md - Documentation development guide
- DOCUMENTATION_SUMMARY.md - Complete summary
- .github/workflows/docs.yml - Auto-deploy workflow
- docs/ - Complete documentation source files

Documentation Deployment:
- Automated deployment to GitHub Pages on push to main
- Manual deployment: mkdocs gh-deploy
- Local preview: mkdocs serve
- Live at: https://zayan-mohamed.github.io/secscan/

Version: 2.1.0
